About This Project

This MBTI Personality Predictor uses two distinct approaches to determine your Myers-Briggs Type Indicator personality type:

1. Text-Based Analysis

Our machine learning models analyze your writing style using natural language processing techniques. Four separate binary classifiers (one for each MBTI dimension) process your text and predict your personality traits based on linguistic patterns.

  • Uses TF-IDF vectorization for feature extraction
  • Separate scikit-learn models for each dimension
  • Trained on MBTI personality data
  • Provides probability-based confidence scores

2. MCQ-Based Test

A deterministic scoring system based on multiple-choice questions. Each question belongs to exactly one MBTI dimension, and each option maps to a specific trait with a score value. The system maintains four independent scores and calculates confidence based on response consistency.

  • Questions stored in JSON format for easy maintenance
  • Deterministic scoring algorithm
  • Transparent and explainable results
  • Dimension-wise confidence calculation

MBTI Dimensions Explained

E/I - Extraversion vs Introversion

This dimension describes where you get your energy from. Extraverts (E) gain energy from external interactions and activities, while Introverts (I) recharge through solitude and internal reflection.

S/N - Sensing vs Intuition

This dimension describes how you process information. Sensing (S) types focus on concrete facts, details, and what is observable, while Intuitive (N) types focus on patterns, possibilities, and abstract concepts.

T/F - Thinking vs Feeling

This dimension describes how you make decisions. Thinking (T) types prioritize logic, objectivity, and consistency, while Feeling (F) types prioritize values, harmony, and the impact on people.

J/P - Judging vs Perceiving

This dimension describes how you approach the outside world. Judging (J) types prefer structure, organization, and closure, while Perceiving (P) types prefer flexibility, spontaneity, and keeping options open.
